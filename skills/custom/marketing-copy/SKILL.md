# marketing-copy

Generate high-converting marketing copy for Jay's SaaS apps.
Email sequences, landing page copy, ad copy, social posts — all tuned for conversion.

## Usage

```
Write marketing copy for: <app-name> [type: email|landing|ad|social] [goal: trial|upgrade|reactivate]
```

## What It Generates

### Email Templates
- **Welcome** — First email after signup, set expectations, drive first action
- **Trial Day 3** — Check in, highlight key feature, reduce friction
- **Trial Day 7** — Show value delivered, social proof
- **Trial Day 12** — Urgency — trial ending soon
- **Trial Day 14** — Last chance, offer incentive
- **Post-Trial** — Didn't convert? Stay warm, drip value
- **Win-back** — Lapsed users, re-engage with new features

### Landing Page Copy
- Hero headline + subheadline
- 3 benefit bullets
- Social proof section
- CTA button copy
- FAQ section

### Ad Copy
- Google Ads headlines (30 chars max)
- Facebook/Instagram ad copy
- Twitter/X posts

## Copywriting Principles Built In
1. **AIDA** — Attention, Interest, Desire, Action
2. **Pain → Solution** — Lead with the problem, present the fix
3. **Specificity** — Use real numbers, real prices, real outcomes
4. **Urgency** — Trial deadlines, limited-time framing
5. **Social proof** — Placeholder for real testimonials
