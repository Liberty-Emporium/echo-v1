# Skills Folder

Skills are located in `/usr/local/lib/node_modules/openclaw/skills/` on the running system.

## Skill Structure

### 1. Single .skill File
```
skills/
  api-design.skill   # YAML file with description
```

### 2. Folder with SKILL.md
```
skills/
  github/
    SKILL.md         # Description and metadata
```
