# Liberty Emporium Web Development Skill

## Purpose
Handle web development projects for Liberty Emporium — build, deploy, and manage web applications.

## Triggers
- "build app", "create website", "new project"
- "deploy", "railway", "push to production"
- "fix bug", "debug", "issue with app"
- "check status", "app health", "deployment"
- "Liberty Emporium web", "web app", "SaaS"

## Capabilities

### Project Setup
- Create Flask/React/Next.js projects
- Set up GitHub repos
- Configure Railway deployment
- Add monitoring (Healthchecks, UptimeRobot)

### Deployment Workflow
1. Code locally or in editor
2. Push to GitHub
3. Railway auto-deploys
4. Verify with health checks

### Troubleshooting
- Check Railway dashboard for errors
- Review deployment logs
- Verify environment variables
- Test database connections

### Monitoring
- Healthchecks for uptime monitoring
- GitHub Actions for CI/CD
- Railway metrics for performance
