# Module 1 — The Foundation
## "Understanding AI Agents & Finding Your #1 Opportunity"

*Duration: ~2.5 hours | 4 Lessons*
*No build required — this is mindset and strategy*

---

## 🎯 Module Goal
Student leaves knowing:
- Exactly what AI agents are (and are not)
- The difference between a chatbot and an agent
- Which part of THEIR business to automate first
- The full toolset they'll use in this course

**Module Deliverable:** A completed "Business Automation Map" — a 1-page document identifying their top 3 automation opportunities, ranked by impact and effort.

### Lesson 1.2 — Agents vs. Chatbots vs. Automation
**Core Message:** Chatbots answer questions. Agents take action. That's the difference.

**Script Topics:**
- A chatbot: responds, then stops
- An agent: responds, then does something (sends email, updates database, books appointment)
- Automation (Zapier/Make): follows rigid rules
- Agent: flexible, thinks, decides
- The sweet spot: agents + automation working together

**Slide Outline:**
1. Title: "Agents vs. Chatbots vs. Automation"
2. Chatbot diagram — input → response → done
3. Agent diagram — input → think → act → report back
4. Automation diagram — trigger → rule → action
5. Side-by-side comparison table
6. Real example: AI Agent Widget (Jay's app) — live demo screenshot
7. "What can an agent DO for YOUR business?" — teaser for Lesson 1.3

**Follow-Along Guide:**
- Visit: https://ai.widget.alexanderai.site — see a live agent in action
- Notice: it doesn't just answer, it remembers context and can take actions
- Write down: "What would I want MY agent to do that isn't just answering?"
