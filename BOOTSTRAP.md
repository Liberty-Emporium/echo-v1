## ⚡ Full Bootstrap Script

Run `/root/.openclaw/workspace/echo-v1/scripts/bootstrap.sh` after a fresh boot.
It will:
1. Install pip, bandit, ruff, playwright, pytest, black
2. Set up /root/.secrets/ directory
3. Configure git identity
4. Clone **echo-v1 only** (the brain repo — not all apps)
5. Set up GitLab backup remote on echo-v1
6. Check EcDash dashboard health + echo-bridge queue
7. Sync brain files (MEMORY.md, SOUL.md, IDENTITY.md) to EcDash dashboard

> Note: Other repos are cloned on-demand when needed for a task.
